    This widget supports two styles of streaming video using UV4L WebRTC or using Motion. WebRTC uses H264 compression so it is more intensive on the server's CPU and less intensive inside ChiliPeppr. Motion uses MJPEG so it is less intensive on CPU for the server, but a bit more intensive inside ChiliPeppr.
